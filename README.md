# extras

## 📥 Installation:

### Install [uv](https://ollama.com/download) for linux:
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Clone the repository:
```
git clone https://github.com/ashuguptahere/extras.git
cd extras
```

## Install all `uv` dependencies:
```
uv sync
```
